## Features
- **Aesthetic UI:** Cute fonts and emoji accents, interactive buttons.
- **Group tools:** Add/kick/promote/demote, tagall, hidetag, setname, setdesc, mute/unmute, linkgroup, revoke, delete.
- **Protections:** Antilink, antibot notice, antispam, antidelete, antiviewonce.
- **Main:** Menu, allmenu, info, ping, owner.
- **Owner-only:** self/public modes, setppbot, restart, shutdown, backup, block/unblock.

## Requirements
- **Node.js:** 20+
- **Login:** Scan QR on first run.
- **Images:** Put your `media/menu.jpg`, `media/welcome.jpg`, `media/qris.jpg`.
